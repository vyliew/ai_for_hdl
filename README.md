# ai_for_hdl

An integrated environment for exploring the intersection of **Artificial Intelligence** and **Hardware Description Languages**. This repository hosts the official source code and interactive curriculum for the *AI-Powered Verilog/VHDL Learning Course*.

## 📌 Overview

This project bridges the gap between traditional RTL design and modern AI workflows. By leveraging Large Language Models (LLMs) and Python-based automation, we aim to accelerate the hardware development lifecycle—from specification to synthesis.

---

## 📁 Repository Structure

The repository utilizes a dual-branch strategy to keep source code and interactive learning environments distinct:

* **`main` Branch**:
* **HDL Source**: Optimized Verilog and VHDL modules.
* **Testbenches**: SystemVerilog and VHDL test environments.
* **Scripts**: Synthesis and simulation automation (Tcl/Shell).


* **`colab` Branch**:
* **Interactive Notebooks**: `.ipynb` files designed for Google Colab.
* **AI Prompts**: Structured templates for LLM-based code generation.
* **Tutorials**: Step-by-step guides on AI-assisted hardware debugging.

## 🚀 Getting Started

### Prerequisites

* **Simulators**: [Icarus Verilog](https://www.google.com/search?q=http://iverilog.icarus.com/), Vivado, or ModelSim.
* **Python**: Version 3.8+ (for local notebook execution).
* **Cloud Access**: A Google account to run notebooks via Colab.

### Installation & Usage

1. **Clone the repository:**
```bash
git clone https://github.com/yourusername/ai_for_hdl.git
cd ai_for_hdl

```


2. **Access the HDL Source:**
The production code is available immediately on the `main` branch.
3. **Access the AI Course Material:**
Switch to the dedicated notebooks branch:
```bash
git checkout colab

```
